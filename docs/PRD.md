# PRD — webchemist.net v2

## 제품 개요

**제품명**: webchemist.net v2  
**목적**: 잠재 클라이언트가 webchemist의 작업물을 확인하고 연락으로 이어지게 하는 브랜드 사이트  
**핵심 전환 목표**: 방문 → 프로젝트 확인 → 연락(이메일/양식/LinkedIn)

---

## 타겟 방문자

**주 대상**: 잠재 클라이언트 (외주/프리랜서 의뢰를 고려하는 사람)

- 사이트에서 원하는 행동: **프로젝트 결과물 확인** 후 신뢰 형성 → 연락
- 브랜드 인상: **신뢰할 수 있는 전문가** — 능력과 경험이 입증된, 믿고 맡길 수 있는 파트너

---

## 페이지 구성 (섹션 순서)

| 순서 | 섹션 | 목적 |
|------|------|------|
| 1 | Hero | 첫인상 — 이름, 직함, CTA(프로젝트 보기) |
| 2 | Projects | 실제 클라이언트 작업물 위주 결과물 열거 |
| 3 | About | 배경, 현재 집중 분야, 작업 방식 |
| 4 | Skills | 기술 스택 |
| 5 | Contact | 연락 채널 — 이메일, 커뮤니케이션 양식, GitHub/LinkedIn |

---

## Projects 섹션 전략

- **실제 클라이언트 작업물 중심** — 무엇을 만들었는지 결과물/성과 위주로 서술
- 각 프로젝트: 이름, 한 줄 설명, 사용 기술, 링크(또는 비공개 표시)
- devist는 자체 프로젝트로 별도 표시 가능

---

## Contact 채널

- 커뮤니케이션 양식 (이름, 프로젝트 내용 입력 폼)
- GitHub / LinkedIn 링크

## 디자인 원칙

- **흑백 미니멀리즘**: 배경 #0f0f0f, 텍스트 #f5f5f5
- **신뢰감 우선**: 과도한 장식 없이 작업물이 스스로 말하게
- **타이포그래피 중심**: Inter + JetBrains Mono
- **단일 페이지 스크롤**: 앵커 링크 기반 탐색
